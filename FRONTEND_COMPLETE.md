**前后端数据流**```用户操作 → 前端组件 → API 请求 → 后端控制器       ↓              ↓              ↓   表单验证 →   Axios 请求 →   服务层处理       ↓              ↓              ↓   状态更新 →   响应处理 →   数据库操作       ↓              ↓              ↓   UI 更新 →   数据展示 →   返回结果```---## 🎯 **核心交互流程**### 登录流程1. 用户输入用户名密码2. 前端验证表单3. 调用登录 API4. 后端验证并返回 Token5. 前端保存 Token 到 localStorage6. 获取用户菜单权限7. 跳转到首页### 权限控制1. 路由守卫检查 Token2. 检查用户权限3. 动态生成菜单4. 按钮级权限验证5. 无权限时隐藏或禁用### 数据管理1. 页面加载时获取数据2. 用户操作触发 API 调用3. 后端处理业务逻辑4. 数据库持久化存储5. 前端更新显示状态---## 📊 **前端性能优化**- ✅ Vue 3 Composition API- ✅ 组件懒加载- ✅ 前端启动失败```bash# 清除缓存重新安装cd frontendrm -rf node_modules package-lock.jsonnpm installnpm run dev```### API 调用失败```bash# 检查后端服务是否运行curl http://localhost:3000/api/auth/login \  -X POST \  -H "Content-Type: application/json" \  -d '{"username":"admin","password":"admin123"}'```### 权限问题- 检查用户角色和权限分配- 查看浏览器开发者工具的网络请求- 确认 Token 是否有效---## 📈 **扩展和定制**### 添加新页面1. 在 `views/`
